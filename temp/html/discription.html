キーマップの選択

Blender デフォルトのキーマップ
Blender 27x バージョン2.7xに基づいた古いキーマップ
業界互換キーマップ Mayaなどに準拠したキーマップ

Select with Mouse Button

「選択」に使うマウスボタンを左右のどちらかに設定

この設定を変えると、他の項目のマウスボタン操作も左右逆になるものが多い。現バージョンでのデフォルトは［左］。

Spacebar Action

Spaceキーの動作を設定

再生 再生
ツール ツール
検索 検索

例： ［ツール］または［検索］を選ぶと、［アニメーション再生］ は Shift + Space になる。

Tool Keys

G（移動）R（回転）S（拡大縮小）などのキーを押したときの動作を設定

Immediate 動作をすぐに有効にする

例： Rを押してそのままマウスを動かすと、マウスボタンを押していなくても選択アイテムが回転する。

アクティブツール 動作に対応したツールに切り替える

例： Rを押すと回転ツールに切り替わる。選択アイテムやギズモをドラッグすると回転する。

Select All Toggles

A

Grave Accent / Tilde Action

`または @ キーの動作を設定

ナビゲート ナビゲート（視点切り替え）のパイメニューを表示する
ギズモ ギズモ（変形操作のカラーガイド）のパイメニューを表示する

Middle Mouse Action

マウス中ボタンの動作を設定

周回 視点の回転
パン 視点の平行移動

ここで選択しなかった方は Shift+マウス中ボタン の動作になる。

Tab for Pie Menu

Tab と Ctrl-Tab の動作を入れ替える

プリファレンスを保存

設定を保存する

［プリファレンスを自動保存］にチェックしておくと、変更した内容が自動で保存される。
